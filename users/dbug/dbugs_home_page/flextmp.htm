MANUALS




This book is the Oric 1 Manual, by John SCRIVEN.
It's a good book, but less complete than the Atmos manual.
The left one is the original UK edition, and the right one is the French version.

This book is The COMPANION to the ORIC 1, by Ian Adamson.
It's really a great book.
In this book you can find how plug and use your Oric Computer (nice !), and you get a brief description of how a computer works.
Next chapters learn you the way of programming in Basic language, to do graphics and sounds.
At the end, you can find an introduction to machine code, and many appendices covering ASCII tables, escape codes, error messages, memory map, MCP 40 printer commands, I/O connectors, and many other interesting stuffs.

It will become the official reference manual sold with Atmos computer (see below)


The Black manual is the Atmos version of "the Companion" The Pink book is the french version of the black manual. The translation is very good. And the white add on covers some mising topics.

The last book is the Swedish version of the Oric Atmos manual by Peter Jonsson.
It is a guide for first time users and it's very thin, only 75 pages. But it is very well written and covers most of the chapters from the english manual.

The Guide Pratique de l'Oric was sell with the Oric 1 in 1983 as an Oric manual. (Edited by Cedic/Nathan)
Authors are Michel Bussac and Robert Lagoutte, with pictures by Dominique Carrara
It talk about the Oric's usage, then go to the Basic explanations. Really a good book for beginers. Gives syntax and examples for every functions. And the pictures are interesting too.



DISK OPERATING SYSTEM




This books is the reference manual for using the Tran Disk Operating System
The TDOS is the DOS that came with the JASMIN drives.
Seems to cover clearly all the disk domain, including what's formating, filesystem, using assembly language, sorts, file access, and so on...

ADVANCED PROGRAMMING




This book (Au coeur de l'Oric Atmos) is my favorite Oric book so far. Mr Gilles Bertin gives very interesting informations.
Topics covered are an hardware description of the Oric, how to create some weird graphics mode mixing High and Low resolution, How to scan the keyboard (using the FUNC key), to program a real time clock, read joysticks, modify datas in basic, using floating point code in assembly language,.... and finaly a near complete disassembly of the ROM !
Great no ?

A book from Andre Cheniere. Dominique Pesant think it's a very good book, even better than "Au coeur de L'Oric Atmos" !
225 interesting page. Edited in 1984 by IS EDITIONS (from Angers, cool ! It's were i'm born :)

The second book is The Oric Programmer by S.M.Gee and Mike James.
Not as comprehensive as the advanced user guide but many small tips if you want to be a better Oric-programmer.

A book from Frederic Blanc and Francois Normant. (Edited by ASN diffusion)

It's a very small book, but with enough descriptions to turn around the Oric and get good time doing poke stuff, and 6502 coding.



6502 PROGRAMMING




THE book on the 6502 programming !
Rodnay Zaks reference book on the 6502. Nothing more to say. If you don't have it, you're missing a fondemental book !

A good book for starting 6502 programming on the oric.
Nothing special, but very progressive, and interesting remarks concerning automodification, relocation...

A nice book from Ian Sinclair, with nice paper, nice presentation, easy-to-understand explanations.
Nothing about clock-cycles, but it's a pretty good book for starting 6502 programming.
Some interesting considerations about anti-piracy tricks.

And another french book that can helps you to learn 6502.
I really do not like it, since it's full of misstyping. And some "dirty code".
I'm ok to use self-modifying code, but in a book you have to explain that this kind of code cannot be executed in ROM.
And the author is happy to use some stupid features like "opcode" overloading.
(That means that depending of where you start to disassemble, the code have different meanings).
Somewhere disturbing. I say.



PROGRAMMING IN BASIC




A french book from Michel Bussac (edited by Cedic Nathan)
A kind of user manual, but talks mainly about graphics and sound commands.

Ok. It's only one of the many books that offers some basic programs to type. Nothing special to say.

The first one, by Jacques Boisgontier and Sophie Brebion is a kind of Oric Basic programming manual.
It's well written, and very progressive. Many sample programs doing graphics animations and so on....

The second one, of the same author, is an extension of the previous one. With programs for exercise, education, gestion, games...

This book is not from Philippe Bayvejiel (or BAYJEVIEL !), as written on the cover page...
The authors are Michel Birades, Dominique Joly, Philippe Le Bitoux and Jean-Marie Vallade.
You can consider this book as an Oric Basic user manual.

After teatching you 6502 programming, Rodnay Zacks teaches you Basic Programming.
As all Zacks book, it's a "multi-platform" book. It uses standard Basic, and exists for the AppleSoft basic, and C64.
Edited by Sybex in 1984.

Some English books for learning programming
The first book is from Peter Shaw, and edited by Virgin.
Many little basic games, but there is a 25kb adventure game !

The second (Sixty programs for the Oric-1) by Robert Erskine & Humphrey Walwyn.
Many small basic programs, most of them are pretty good. From games like Alien to Bioryth-program. A good book for a starter.

Three french books on Oric Basic programming.
"Des programmes pour votre Oric", from Michel Piot.
24 little basic programs (16kb Oric compatible) that could be used as a Basic learning class.
An explanation of used instructions alows to understand these programs.

"L"Oric à l'affiche" from Jean-Francois Sehan (PSI editions).
20 programs, with algorithms and all needed explanations.

"Jeux sur Oric" from David Chane-Hune and Francois Darbois (Edimicro editions).
Another initiation book.

Three English books on Oric Basic programming.
Getting started on your Oric, from Tim Hartnell and Peter Shaw.
A very little book, a kind of mini-user-manual. Fit in a pocket.

The ORIC-1 and how to get the most from it, from Ian Sinclair.
This is the longest title for an Oric book.
Nothing special at all.

20 games for the ORIC-1, from Wynford James.
20 programs, with algorithms and all needed explanations.



TELESTRAT MANUALS




We enter now in the world of the Telestrat (or Stratos...).

The first book is the Hyper-Basic manual, written by Fabrice Broche and George El Andaloussi.
Nothing special to say. It's only an index of the Basic Key Words.

The second one, the Manuel des applications télématiques, written by Pascal Cusset, concerns the "Telematic" cartridge. It's mostly for French Users that wish to create a "minitel" server.

HARDWARE




Theses books are for those that want to use their soldering iron :)
The first one is dedicated to the communication.
The second one learns you electronics, using your Oric.
The third one explain how to make expansions boards.

From Micro systemes edition, by R. Schulz It covers system's architecture, hardware's problems of both Oric, interfaces, extensions, etc...
If think it's a good book for hardware people who want to do some Oric's extensions.
