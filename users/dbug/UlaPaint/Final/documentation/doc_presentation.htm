Introduction

What is UlaPaint ?

First thing: This is *NOT* a Painting Program for the PC !

UlaPaint is an "easy to use" (well, it's the original idea) painting program for creating pictures for the Oric computers. Oric is an English brand of 6502 based 8 bits computers. This software can be used either on the real hardware, or on an emulated machine.
