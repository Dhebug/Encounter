What is an attribute ?

In Hires mode, any byte in memory that happens to have a value in the range [0-31] or [128-159] is considered as an attribute. The second interval simply described the fact that the video inverse bit (value=128) is active.

Values that are not in this range are simply pixels patterns that are used to draw on screen at pixel level.

Of course, since It's not possible to put TWO values in the same byte, it's not possible to have at the same location an attribute and some pixels. If you use an attribute somewhere on screen, this area will appear as 6 horizontal pixels using the background color.

6 Change INK to CYAN
7 Change INK to WHITE
