Version history

Version 0.010 - september 5th 2003

Fixed bugs:
  • Romu: Keys 'ASDFGH' were not working anymore in byte mode
New features:
  • Current coordinates are now displaye on the left of the screen
  • The 'Q' key gave access to the tool menu selection

Version 0.009 - september 3nd 2003

Modifications
  • Added a lot of content into documentation
  • Moved invert pixel key from 'Z' to '-'
  • Moved invert video key from 'X' to '='
  • Mode 'draw none' previously accessed by key 'Q' has been suppressed
New features:
  • Memorisation of current picture into an undo buffer with the key 'Z'
  • Swap between undo buffer and current picture with the key 'X'
  • An icon in the information bar now remember the current tool

Version 0.008 - september 1st 2003

Modifications
  • Pressing keys 1 to 8 change the "memorised byte" to an Ink color attribute
  • Pressing keys 1 to 8 + LSHIFT change the "memorised byte" to a Paper color attribute
  • Documentation improved (?) and changed to HTML with pictures and links
  • In BYTE mode, you now paint with the currently memorised byte

Version 0.007 - august 31 2003

Modifications
  • Informations in the text lines are probably now a lot more understandable
New features:
  • Added a "protect attribute" functionality.
  • New LOAD/SAVE of a file called TEST.TAP
Optimisations:
  • Totaly new keyboard handling, things should be now a lot more responsive

Version 0.006 - august 31 2003

Fixed bugs:
  • Romu found a bug in fullscreen display of version 0.005

Version 0.005 - august 31 2003

Fixed bugs:
  • Ooup, version 0.004 didn't displayed anymore informations in the text lines... corrected

Version 0.004 - 31 2003

Optimisations:
  • A lot of routines have been switched to assembly code because the C was too slow
  • TEXT and HIRES routines are now custom, means that there's no more flickering when entering or leaving menus.

Version 0.003 - august 29 2003

Modifications:
  • Now using a 8 key pad to move the cursor. That allows diagonals, a lot better
  • Two non shifted/controlled key to resize the zoomer
  • Space to switch between fullscreen picture and zoomed mode
  • The cursor is now thicker, for a better visualisation
New features:
  • In pixel mode, the cursor show only one pixel
Fixed bugs:
  • The step of movement is now different in pixel and byte mode
Optimisations:
  • The erasing of buffer is now done in assembler, no more temporisation at begining

Version 0.002 - august 27 2003

Modifications:
  • ESCAPE instead of SPACE to reach the main menu
New features:
  • A cursor indicating the edited byte is displayed and correctly moving with the view
  • SDFGHJ keys to switch individual pixels of the byte
  • ENTER switch the current pixel
  • I invert the current byte
  • C and V to COPY and PASTE a byte from screen to screen
  • Buffer initialised to 64 instead of 0
Fixed bugs:
  • Video inverted INK change was not displayed correctly.

Version 0.001 - august 24 2003

New features:
  • real time resizable zoomer with two modes: colored/codes
  • integrated help pages
