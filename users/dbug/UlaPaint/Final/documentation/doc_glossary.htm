Glossary

ATMOS
Evolution of the Oric 1, that mostly has a debugged system, and a real mechanical keyboard.


Attribute
A special code that appear in the video memory that affect the display. Attributes can be used to switch between HIRES and TEXT modes, change refresh frequency, change colors, enable blinking, getting access to special text display functions like double height or alternate character set.
Each attribute occupies one byte.
See also the Attribute table.

Byte
The smallest physically adressable element of the computer memory. A byte is simply a memory cell. The 64k Oric memory is made of 65536 bytes. The Hires video memory occupies 8000 bytes. A Hires scanline is made if 40 bytes.
Internaly, a byte contains 8 bits. A byte in video memory can represent a character code, a pattern of pixels to display on screen, or an attribute code.

Character
A character occupies one byte in memory, and is used to display letters, numbers, punctuation signs...
Each character is made of a matrix composed of 6x8=48 pixels.

HIDE
A painting program made by Jonathan Bristow
Hires
One of the two video mode of the Oric.
The HIRES (High Resolution) mode is composed of 200 lines of 240 pixels, and also has an additional 3 lines of 40 characters that matches the 3 last lines of the TEXT mode.

Ink
By nature the Oric video display is monochromatic. Background pixels are considered as PAPER while foreground pixels are considered as INK. By default at the begining of each scanline the PAPER is set to BLACK, and the INK is set to WHITE. It's possible to change the color of INK by inserting an INK CHANGE attribute.

Lorigraph
One of the few good paint program that has been edited on the Oric by Loriciels
Master Paint
One of the few good paint program that has been edited on the Oric by Ere Informatique
Matrix
A Matrix is simply a way to tell that something is made of X by Y elements stored in a rectangular fashion. The Hires video memory is a 240x200 matrix, a character is made of a 6x8 matrix, etc...

Microtan 65
The ancestor of the Oric. Advanced versions had an hexadecimal keypad to enter commands, and very few memory.

Oric 1
First real Oric computer, that succeeded the Microtan 65.


Paper
By nature the Oric video display is monochromatic. Background pixels are considered as PAPER while foreground pixels are considered as INK. By default at the begining of each scanline the PAPER is set to BLACK, and the INK is set to WHITE. It's possible to change the color of INK by inserting an INK CHANGE attribute.

Pixel
Pixel is the contraction of pictures element. It describes the smallest individually addressable elements that compose a picture.
In Hires the screen is made of 240x200=48000 pixels.
In Text, each character is made of a matrix composed of 6x8=48 pixels.

Raster
A rasterline is originaly equivalent to a scanline. It's now often used to refer as the fact of changing the colors of the background on screen in a way that it propagates to the end of the current line. By changing this color at the begining of each line, we get what is called a "raster bar".
Often used in demos.

Resolution
The resolution simply defines the number of individualy adressable elements of the picture displayed on screen.
In Hires mode, the resolution is of 240x200 pixels.
In Text mode, the resolution is of 40x28 characters.

Scanline
A scanline is simply the term used to describe the movement of the electron bean moving horizontaly across the screen to perform the display. It's also sometimes called 'raster'. By extension it's often used to indicate we are talking of an horizontal line in the video memory. On the Oric, a scanline contains 240 pixels.

Telestrat
The Telestrat is the last model of the Oric family. It provide a serial interface, two joystick port, and integrated floppy disc controlleur, ROM cartridges slots, a second VIA and banked memory.
Text
One of the two video mode of the Oric. The TEXT mode is composed of 28 lines of 48 characters.

ULA
A dedicated chip designed by Oric International that is responsible of synchronizing all the other chips that are into the machine, but also perform video display.
