Known issues

Performance

This program is still a beta version, not at all a final release. Of course I try to keep the speed to something reasonable to make it usable, but I will not concentrate on speed/performance for the moment, I will wait for the feature set to be there.

Keyboard reaction

The keyboard handling probably need to be rewriten, but it's a hard part, so I will keep that for later while it's not a major pain to use it.

Keyboard layout

Ok, the keyboard layout is probably not intuitive if you use a PC. But it's fine for an Oric Atmos. Can you live with that for the moment ?

Load and Save

For the moment there is no way to load or save picture on Floppies, and you cannot enter the name you want. Of course it will be changed one day, but for the moment the "test.tap" filename should be enough for testing, no ?

Feature set

Yes, you cannot draw lines, circles, fill areas, type text, put colors...
It will of course be possible in the future, but not for the moment... be patient !

Compatibility

Since UlaPaint is performing Rom calls for files and keyboard handling, it's working only on the Oric Atmos. Making it work on the Oric 1 or Telestrat imply to either replace the rom calls by equivalents on these machines, or rewrite the faulty code.
