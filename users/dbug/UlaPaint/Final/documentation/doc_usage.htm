Usage

The user interface

This program has two separate modes:

  • Menu mode
  • Editing mode

To switch from Menu to Edit, and vice versa, use the ESCAPE key.

The menu mode

To navigate into the menu, simply press on the indicated keys.

Note: For number, on an emulator you will perhaps have to press the SHIFT key also.

The editing mode

In editing mode, the screen is separated in three parts:

  • The picture, at 1x1 scale
  • The zoomer, at 6x6 scale
  • The informations (3 lines of 40 characters)

It's possible to see the picture in whole (suppressing the zoomer) by pressing the space bar.

Pressing it again will bring back the zoomer.

For precise work on a picture that has uses complex things like inversion and colors attributes, it's possible to display the value of each byte in hexadecimal in the zoomer.

The information

On the right of the screen are written three block of text.

  • The first indicate the display mode of the zoomer: BYTE or PIXEL
  • The second indicate the drawing mode: WRITE, ERASE or REVERSE
  • The third one indicate if the attributes are protected while drawing, or not.

On the left side are simply the coordinates of the cursor.

Also an icon indicate the tool currently in use... so far only the hand drawing, but some other are planned:

  • - hand drawing
  • - lines
  • - rectangles
  • - filled rectangles
  • - ellipses
  • - filled ellipses
  • - flood fill
