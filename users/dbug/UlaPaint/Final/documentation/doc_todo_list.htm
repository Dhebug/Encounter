To do list

Feature requests

If you want to ask for a feature that is not present in the following list, you should go to www.oric.org, in the "community" section, and post a message in the "UlaPaint" forum. This way, everybody will be able to see it, discuss about it, and eventually propose alternative things.

Planned features

Here is the list of planned features (in no particular order):

  • Advanced drawing primitives likes Lines, Rectangles, Circles, ... filled or outlined
  • Support for font loading/saving, and possibility to type text into a picture
  • Bloc copy, symetry, shifting, cloning, rotations, ...
  • Full undo/redo
  • Possibility to have more than one picture in memory at the same time
  • Flood fill
  • Drawing with brushes (or patterns)
  • Simple sprite animation previsualisation
  • Packed picture format
  • Full disk/tape load and save support, with directory, name requester
  • cut a sprite and save it
