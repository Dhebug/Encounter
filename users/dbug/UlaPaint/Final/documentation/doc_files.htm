Load and saving files

For the moment it's not possible to load a specificaly named file, or save a file with a specific name. For testing purpose you can only load one of the 6 pictures that came in this ZIP distribution, or LOAD/SAVE a file called 'TEST.TAP'.

To access the file menu, just go to the main menu (press ESCAPE if you are in editing mode), then select the third option "buffer operations", and then choose any of desired option. Just be sure of what you are doing, because you will not have any confirmation box ;)

Format of files

UlaPaint only load and save pictures of exactly 8000 bytes. If the file is longer, it will be truncated. If it's shorter, I don't know how it will react :)
