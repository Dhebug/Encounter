Installation

Configuring UlaPaint

Just unzip everything at the place where the emulator expect to find Tapes.

It will be probably either at the root of the emulator folder, or in an "tapes" folder.

Note for linux users:

With xeuphoric user can unzip it anywhere but that they must run xeuphoric from the directory in which they have unzipped the files.

Running it

Just double click on "ulapaint.tap", or use CLOAD"ulapaint" from the oric.

If you are using linux, take care of the fact that the systeme is case sensitive...
