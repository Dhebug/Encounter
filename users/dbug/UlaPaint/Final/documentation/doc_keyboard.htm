Keyboard

Why using the keyboard ?

The Oric got only very few peripherals that could be use to help drawing on the machine. The only available graphic tablet (Graphiscop) was totaly unusable due to its design and of the poor accuracy. The lightpen is usable, but only on non dark areas, and it is very painful to use for a long period of time. And finaly the only machine equiped of a mouse was the Telestrat, but almost no one has such a peripheral.

The keyboard is the sole remaining usable device for our purpose.

Moving the cursor:
	[I][O][P]
	 [K]   [']
	  [,][.][/]

	  Note: pressing shift will move faster
	  Note: pressing control when moving will draw a pixel/byte

Color changes (switch to byte mode):
	[1][2][3][4][5][6][7][8] set ink from 0 to 7

	Note: pressing LSHIFT at the same time will set the color of paper.


Direct pixel setting (mostly in byte mode):
	[A][S][D][F][G][H]  set one of the 6 pixels of the byte
	[L]					set current pixel

Mode Switching
	[ENTER]      switch between pixel mode and byte mode
	[W][E][R]	 switch between WRITE, ERASE, and REVERSE mode

Byte manipulations:
	[-] invert pixels
	[=] invert video

Undo - Redo:
	[Z] memorise current picture into the undo buffer
	[X] exchange undo buffer and current picture

Cut and paste:
	[C] copy in memory the byte under the cursor
	[V] put the memorised byte in the location under the cursor

Quit to menu:
	[ESCAPE]

Resize zoomer:
	[UP ARROW]
	[RIGHT ARROW]

	Note: pressing shift will maximise/minimise the zoomer


View fullscreen:
	[SPACE]
