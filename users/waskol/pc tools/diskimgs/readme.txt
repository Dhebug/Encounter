I put there some image disk images you can use with Omniflop utility which permits to Read, format and write Oric floppies from Windows Vista/Seven/2000/etc...

OmniFlop is a ‘universal’ floppy disk reader, writer, and tester by Jason Watton, for the IBM PC or compatible which can handle alien floppy disk formats not normally supported by DOS, Windows and Linux.

Omniflop is freeware.
