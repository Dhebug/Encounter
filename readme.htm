Please behave

If you don't know anything about Subversion, you can learn more about it here:
  • The Subversion website
  • The official SubVersion book
If you want to install a subversion client, here is a selection for the various OSes:
  • Windows: Tortoise SVN (binaries)
  • BeOS, Haiku: SVN binaries
  • GNU/Linux:
    • Debian-based, Ubuntu: subversion package, or apt-get install subversion
    • RPM-based (RedHat, Mandriva, ...): rpm -ivh subversion
  • MacOS: Subversion (binaries), SCPlugin (binaries, installation)
If you want to be added as a user so you can modify, please contact the webmaster.
